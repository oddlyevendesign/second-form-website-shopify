# SECOND FORM — Shopify Project Context

## Brand
- Name: Second Form
- Category: Elevated lounge / hybrid wear
- Positioning: Between Alo Yoga and SKIMS — comfort-first but polished
- Target customer: Urban women 23–38 (also sells menswear)
- Brand feel: soft, calm, confident, effortless, premium

## Dev Principles
- Mobile-first (90–95% mobile traffic)
- CRO-first — prioritize usability and conversions over artistic design
- Fast loading — avoid heavy animations or scroll effects
- Scalable — easy to optimize monthly, supports A/B testing
- Minimize app reliance

## Branded Fabrics
- CloudSoft™ — Pima Cotton + Modal + Spandex. Ultra-soft, breathable, everyday.
- LuxeModal™ — Cotton Modal Spandex. Silky, lightweight stretch.
- AirSoft™ — Micro Modal + Spandex French Terry. Brushed interior, lightweight warmth.
- EcoSoft™ — Recycled Poly + Cotton + Spandex Terry. Soft, durable, eco.
- EcoLite™ — Recycled Poly + Cotton + Spandex Jersey. Breathable, sustainable.
- SculptStretch™ — Nylon Rayon Spandex Roma. Structured, smooth, strong recovery. For fitted pieces.
- TextureRib™ — 1x1 Rib Cotton Poly Spandex. Flexible, textured.
- WaveRib™ — Variegated Rib Cotton Poly Spandex. Textured rib, stylish.
- ElevateKnit™ — Cotton Poly Spandex Ottoman. Subtle structure, durable.
